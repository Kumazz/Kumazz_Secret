# 字符串

### 认识字符串

&emsp;&emsp;字符串是 Python 中最常用的数据类型，用**引号(单引号、双引号、三引号)**来创建字符串，并且赋值给一个变量
&emsp;&emsp;如: a = 'hello'，a = "python"，a = '''world'''、a = """python"""

&emsp;&emsp;**不可变数据类型**，即字符串不可修改

### 字符串属性
*  **输出**
 * 三引号的区别在于，三引号支持换行(不会显示和添加换行符)用于大文本
 
 ```python
    a = '''hello
        world'''
    --------------------------
    >>> hello
             word
 ```
 
 * 单双引号混合写，用于引用或者英文
  
 ```python
    a = " I'm a boy "
    --------------------------
    >>> I'm a boy

 ```
 
 * 斜杠 \ 进行转义
 
 ```python
    a = 'I\'m a boy'
    --------------------------
    >>> I'm a boy
 ```
 
 * 拼接，在字符串中 「 + 」是拼接符
 
 ```python
    a = 'hello'
    b = 'python'
    c = a + b
    print(c)
    -----------------------
    >>> 'hellopython'
 ```


* **下标**，下标又叫**索引**，就是编号，可以通过下标快速找到对应的数据
 * **下标起始值默认从 「 0 」 开始**
 * **语法:** str[ index ]，返回下标对应元素字符串
 ```python
    a = 'hellopython'
    print(a[1])
    -------------------------
    >>> 'e'
 ``` 
 
* **切片**，对操作的对象截取其中一部分的操作
